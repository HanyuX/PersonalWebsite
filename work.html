Theia

Theia is a rendering engine. Water, sphere, mirror and walls are rendered in the scene. There are a lot technologies are used in the engine, including global illumination, path tracing, soft shadow, streaming retracing and normal mapping. This is a open source project and the original code could be found on github

Feb

Icare Assistant System.

Android App, East China Normal University

Icare assistant system help to remind and assist elders to eat medicine. Doctors and crowdsourcing mechanism set medicine schedule for elders together. Bar code and camera supervise them eat medicine on time. Website and text messages give feedback to family members and doctors. The app gets software copyright in China and is on trial in some communities in Shanghai.

Foreground Editing Technique

Research Project, Digital Entertainment Lab

Foreground Editing Technique separates foreground and background of images with little interaction with users. Extracted foreground could be blended into another image automatically. Image matting and illustration techniques are used to refine result.
